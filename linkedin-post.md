I built IntelliWatt AI, an AI-based smart power monitoring dashboard using Python and Streamlit.

The project simulates appliance-wise electricity consumption, displays live power usage, estimates electricity cost, detects overload conditions, and predicts future power demand using a Linear Regression model.

Key features:
- Live power monitoring dashboard
- Appliance-based simulation
- AI-based future usage prediction
- Real-time graph and activity feed
- Overload detection
- Realistic electricity cost estimation

Tech stack:
Python, Streamlit, Pandas, NumPy, Scikit-learn

Currently, the project runs locally on Streamlit. The next step is to connect it with real hardware sensors using Arduino or ESP32 for real-time power monitoring.
